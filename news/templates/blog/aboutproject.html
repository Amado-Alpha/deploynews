News aggregator platform

First things first!. In a nutshell A news aggregator platform is a service or application that collects and curates news content from various sources across the internet and presents it in a centralized location for users. These platforms use algorithms to gather articles, headlines, and other news-related content from multiple websites, blogs, and news outlets. The goal is to provide users with a conveni ent way to access and stay updated on a wide range of news topics without having to visit mu ltiple websites individually.

So this projects does the same, but specifically for functionality demonstration purpose, i have just used one site, the popular USA's news platform The Onion as my platform data source,but this project can be adapted to include multiple news sites.

Technology behind:- The project utilises the powerful user agent for web crawling called googlebot, and beautiful soup for pulling data out of HTML and XML files. The project also uses celery tasks and beat to improve users experience by performing scheduled web scraping without users interaction, this means that even if a user does'nt search for new updates, the site has the capab ility of doing that for him in specified time intervals. Finally the project as whole is maily built using django. Thank you a lot! for taking your time to have a loot at my work.
